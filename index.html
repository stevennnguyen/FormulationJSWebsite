Form Definition

formDef.json
Form
Field Name Field Type Description
name string The name of the form. Displays at the top of the form.
body string The description of the form. Renders below the form name.
elements Object Array List of form elements that will be rendered in the form.
Element Object
Field Name Field Type Description
type string The type of form element we want to render. Currently supports 'textbox', 'checkbox', or 'dropdown'.
data Object Attributes to render our form element. Object will vary depending on the element type.
Textbox Data Object
Field Name Field Type Description
id string A unique id for the text input.
name string Name attribute for the text input.
value string The initial value of the text input.
label string A label rendered adjacent to the text input.
Checkbox Data Object
Field Name Field Type Description
name string Name attribute for the checkbox input.
value boolean The initial value of the checkbox input.
label string A label rendered adjacent to the checkbox input.
Dropdown Data Object
Field Name Field Type Description
options Object Array An array of the different options in the dropdown element.
value string | int The initial value of the dropdown element. Any option that matches this value will be selected by default when the dropdown renders.
Dropdown Options Object
Field Name Field Type Description
name string Text that will be displayed for a dropdown option.
value string | int The actual value passed when selecting this option.

The form definition is composed of a JSON file with three fields. The first field "name" is a string. This acts as the title of the form. The second field "body" is a string that provides a small description of what the form is for. The third field "elements" holds all the information in order for FormulationJS to render the correct components. "Elements" is a list of indefinite objects.

FormulationJS currently supports three types of components: textbox, checkbox, and dropdown list. These are the objects presented inside of "Elements." All three components are objects that contain two fields: "type" and "data." "Type" is specific to the component name so it will be the name of the component.

"Type" is a string "textbox." The second field is "data," an object that holds "id," "name," "value," and "label" all of which are strings.

"Type" is a string "checkbox." "Data" is an object with a "name," and "label" which are strings, and a "value" which is a boolean.

"Type" is a string "dropdown." "Data" is a list of objects. Each object contains two fields: "name" and "value." "Name" is a string. "Value" is an integer.


{
  "name": "Form Title",
  "body": "Form Description",
  "elements": [
    {
      "type": "textbox",
      "data": {
        "id": "id",
        "name": "name",
        "value": "",
        "label": "label"
      }
    },
    {

      }

        {

        }
      ]
    }
  ]
}

Checkbox

Allow for multiple answers to a questions using checkboxes.


Code


class Checkbox extends React.Component {

  constructor(props) {
    super(props);
    this.state = {
      checked: this.props.checkbox.value
    };
    this._onChange = this._onChange.bind(this);
    this._labelClick = this._labelClick.bind(this);
  }

  _onChange() {
    this.setState({
      checked: !this.state.checked
    });
  }

  _labelClick() {
    this._onChange();
  }
  
  render() {
    var checkbox = this.props.checkbox;
    return (
      <div>
        <label htmlFor={checkbox.name} onClick={this._labelClick}>{checkbox.label}</label>
        <input type="checkbox" checked={this.state.checked} name={checkbox.name} onChange={this._onChange}/>
      </div>
    );
  }
}

export default Checkbox;

Dropdown List

Provide a long list of options to choose from.


Code


class Dropdown extends React.Component {

  constructor(props) {
    super(props);
  }

  render() {
    var options = this.props.options;
    return (
      <div>
        <select>
          {
            options.map(function (option, index) {
              return <option key={index} value={option.value}>{option.name}</option>;
            })
          }
        </select>
      </div>
    );
  }
}

export default Dropdown;
